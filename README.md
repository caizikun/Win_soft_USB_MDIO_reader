Программа под OS Win , на C#
This programm write on OS Win, language (C#). For read (write) CFP module into USB-MDIO interfeises. Use demo board Cypress.

Use registers
0x8000-807f - CFP NVR 1
0x8080-80ff - CFP NVR 2
0x8100-817f - CFP NVR 3
0x8180-81ff - CFP NVR 4

0x8400-847f - Vendor NVR 1
0x8480-84ff - Vendor NVR 2

0x8800-887f - User NVR 1
0x8880-88ff - User NVR 2

0xA000-A07f - CFP module VR 1
0xA080-A0ff - MLG VR 2

0xA200-A27f - Network lane VR 1
0xA280-A2ff - Network lane VR 2
0xA300-A37f - Network lane VR 3

0xA400-A47f - Host lane VR 1

0xAC00-Afff - Common data block
